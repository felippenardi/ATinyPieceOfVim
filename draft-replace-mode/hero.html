When I looked into @csswizardry fancy spaced CSS comment style, I wondered what would be the best way to edit and maintain it.

Using INSERT mode did not sounded a nice idea, as it would require a lot of additional actions to align the columns again.


Native solution: use the replace mode

There is a built-in solution for this: REPLACE mode, where you can insert text and override anything that passes your way. It is entered by typing shift+R. This is what Harry himself suggested.

But if you really fancy this spaced column code style, you might like this suggestion @millermedeiros sent me:


Plugin solution: the Tabular.vim

Tabular.vim, an advanced plugin to help you automate the tabular style code creation. Worth taking a look.
