A fine native CtrlP alternative to search files and folders deep inside the folder structure.

Issue #12

CtrlP is awesome. And so do :e

There is this famous Vim plugin for searching files on a project, the CtrlP, but it is good to know that Vim has a similar built-in functionality:

Gif recorded with LICEcap. Keystrokes preview by KeyCastr.

Get inside command mode and type:

:e **/

Then, complete with the name of the file you are looking for, no matter how deep in the folder structure it is.

:e **/application.css

Hit Tab to search and it will return the first occurrence.

You can also specify any parent directory to refine the search.

:e **/assets/application.css

That is not a replacement for CtrlP, but it is definitely a handy shortcut you will be able to use anytime on any Vim configuration.
