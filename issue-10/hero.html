Using REPLACE mode to maintain column code style.

Issue #10

The Vim REPLACE mode is powerful and often overlooked. Its similar to INSERT mode, with the difference that it will override every next character as you type.

When looking at the column style @csswizardry uses on inuit.css commented summary, I wondered what would be the best way to edit and maintain it.

It turns down that REPLACE mode fits perfectly. Take a look:

You can enter REPLACE mode by typing shift+R


Bonus

If you are interested in get fancy with spaced columns in Vim, you might want to take a look at Tabular.vim, a pluging that makes it easy to use and maintain column code style.
